QUEUES _ COMMANDS

Creation COMMANDS:
crtmqm -ll -lp 200 -ls 100 -lf 1000 DELL ----- To create the QMANAGER along with the log type & log size( in real time for PROD we will prefored)
DEFINE PROCESS(MYPROCESS) APPLTYPE(EXEC) APPLICID('/path/to/app') --- define the process for trigggering 1st then create queue along with process
DEFINE QLOCAL(MYQUEUE) TRIGGER TRIGTYPE(FIRST) TRIGMPRI(0) TRIGDATA('StartMyApp') PROCESS(MYPROCESS) INITQ(SYSTEM.DEFAULT.INITIATION.QUEUE)   ---- queue creation along with triggering inclindig above process


BACKUP_COMMANDS:
***************
dmpmqcfg -m INDUSNET -a -o mqsc > INDUSNET_config.mqsc  --- To take backup of QMANAGER configration ( AT the time of Migration, cloning, backup, )
runmqsc <QMNAME> < INDUSNET_config.mqsc    ------------ To restore the configuration from DC to DR same QMANAGER.
dmpmqmsg -m DEVTIMQMGR -i TI.GWY.OUT -f /IBMMQ/TI.GWY.OUT.msg   --- To take the backup of msg from perticular queue

Permission Commands:
**********************
ALTER QLOCAL(NTS_STP_RES) MAXMSGL(15000000)  --- TO CHANGE PARAMETERS
dspmqaut -m DEVTIMQMGR -t qmgr -p coadmin    ---- TO CHECK USER PERMISSIONS
setmqaut -m DEVTIMQMGR -t qmgr -p coadmin +connect +inq  ---TO PROVIDE PERMISSIONS TO USERS

DISPAY COMMANDS:
***************
DISPLAY QLOCAL(*)  --ALL QUEUES
DISPLAY CHSTATUS QSTATUS(FSR.INCOMING) -- status of QUEUE
DISPLAY QSTATUS((INDBBRQ)) CURDEPTH IPPROCS OPPROCS    --- to check connected of sending & consuming services information
DISPLAY QLOCAL(FSR.INCOMING) MAXDEPTH      ---- To check MAX depth of QUEUE
DISPLAY QMGR SSLKEYR                        ----- To check SSL Configured or not 
DIS CHANNEL(FBCCMTPCH) ALL                    ---- If below to parameters are empty their is no ssl configured to Channel 
[SSLCIPH( )   SSLPEER( ) ]          

Verifiying Channel Auth
************************
DISPLAY CHLAUTH(*) TYPE(ADDRESSMAP)
DISPLAY CHLAUTH(*) TYPE(USERMAP)
DISPLAY CHLAUTH(*) TYPE(BLOCKADDR)
DISPLAY CHLAUTH(*) TYPE(BLOCKUSER)         this commands used to check the Channel  Authuntications.
CLEAR QLOCAL(<TransmissionQueueName>)   ---To clear Pending & unwanted Messages in a QUEUE. 

triggering commands:
***************************
DISPLAY QLOCAL(CBSINDBTRQ) TRIGGER TRIGTYPE INITQ USAGE TRIGDPTH   --- to check trigging enabled or not 
ALTER QLOCAL(CBSINDBTRQ) USAGE(XMITQ) TRIGGER TRIGTYPE(FIRST) INITQ(SYSTEM.CHANNEL.INITQ) ---- to enable the trigging note here INITQ(SYSTEM.CHANNEL.INITQ) is fixed no need to change that value
